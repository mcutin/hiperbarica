# Sistema de Testes Hiperbáricos Automatizados
Este projeto refere-se apenas a um dos componentes de um projeto mais abrangente. Ele destina-se a criar a interface de usuário para configurar, operar e registrar testes hidrostáticos em câmara hiperbárica.

Essa interface será executada em um PC com sistema operacional Windows 7 e/ou 10. Através dela será possível:

1. Definir a configuração dos ciclos de teste
2. Receber as leituras de pressão do sistema
3. Emitir carta gráfica que será anexada à documentação final do teste
